# Loomin

Loomin is a modern study companion built with HTML, CSS, and JavaScript. It features a task manager, focus timer, and integrated AI chatbot to help students when stuck. Designed with custom Google Fonts and a clean UI/UX, it delivers an aesthetic and distraction-free learning experience for the new generation.

# Features

- Task Manager – Add, manage, and track daily study tasks
- Focus Timer – Built-in timer to improve concentration (Pomodoro-style)
- AI Chatbot – Ask questions instantly when you get stuck
- Modern UI/UX – Clean, aesthetic, distraction-free design
- Custom Google Fonts – Unique typography for better visual appeal

# Tech Stack

- HTML5 – Structure
- CSS3 – Styling & layout
- JavaScript (Vanilla JS) – Functionality & logic
- Google Fonts – Custom typography
